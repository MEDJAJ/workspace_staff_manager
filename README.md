WorkSphere – Gestion interactive du personnel
Description

WorkSphere est une application web innovante permettant la gestion visuelle et interactive du personnel au sein des espaces de travail. Elle facilite l’organisation, la répartition et le suivi des employés sur un plan d’étage en temps réel, tout en respectant les règles métiers liées aux rôles et aux zones.

Fonctionnalités principales

Gestion des employés : ajout avec photo, email, téléphone et expériences professionnelles dynamique.

Plan d’étage interactif avec 6 zones :

Salle de conférence

Réception (réservée aux Réceptionnistes)

Salle des serveurs (Techniciens IT)

Salle de sécurité (Agents de sécurité)

Salle du personnel

Salle d’archives

Respect des règles métiers par zone et rôle.

Interface intuitive et responsive avec glisser-déposer, animations et boutons d’action (“+” et “X”).

Limitation du nombre d’employés par zone.

Prévisualisation des profils et photos des employés.

Technologies utilisées

Front-End : HTML5, Tailwind, JavaScript

Gestion de code : GitHub

Design : Interface moderne, responsive, boutons colorés (vert, orange, rouge), formes arrondies

Gestion de projet : Trello

Déploiement : GitHub Pages ou Vercel

Validation : W3C Validator pour HTML et CSS
